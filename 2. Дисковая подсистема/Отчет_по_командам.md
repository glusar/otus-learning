Чистим суперблоки, если они есть (спойлер - нет):  
```bash

```

```bash
lusar@test-vm:~$ sudo mdadm --create --verbose /dev/md0 --level=10 --raid-devices=4 /dev/sd{b,c,d,e}

mdadm: Defaulting to version 1.2 metadata

```

### Чиним RAID
Удалим сбойный диск:  
```bash
lusar@test-vm:~$ sudo mdadm /dev/md0 --remove /dev/sdc
mdadm: hot removed /dev/sdc from /dev/md0
```

Добавим новый диск:  
```bash
lusar@test-vm:~$ sudo mdadm /dev/md0 --add /dev/sdc
mdadm: added /dev/sdc
```

### Создаем GPT таблицу, разделы и монтируем их
```bash
lusar@test-vm:~$ sudo parted -s /dev/md0 mklabel gpt

lusar@test-vm:~$ sudo parted /dev/md0 mkpart primary ext4 0% 20%
Information: You may need to update /etc/fstab.

lusar@test-vm:~$ sudo parted /dev/md0 mkpart primary ext4 20% 40%
Information: You may need to update /etc/fstab.

lusar@test-vm:~$ sudo parted /dev/md0 mkpart primary ext4 40% 60%        
Information: You may need to update /etc/fstab.

lusar@test-vm:~$ sudo parted /dev/md0 mkpart primary ext4 60% 80%        
Information: You may need to update /etc/fstab.

lusar@test-vm:~$ sudo parted /dev/md0 mkpart primary ext4 80% 100%    
Information: You may need to update /etc/fstab.
```
